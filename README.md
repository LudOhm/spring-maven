# Cours de M1 Cybersécurité

## Présentation de l'atelier :

Afficher une page web en java directement via une image poussée sur Dockerhub.

_Le projet fonctionne avec Docker et Springboot-maven_

### fichiers : 

- ensemble de springboot-maven télécharger en version 4.0.2

- Dockerfile

- deploy.yml pour faire l'aciton github et se connecter à Dockerhub

- DemoGithubApplication.java pour avoir le code à afficher sur la page web


**Atelier réalisé dans le cadre du cours DevSecOps à EFREI Paris.**

_Année 2025-2026_
